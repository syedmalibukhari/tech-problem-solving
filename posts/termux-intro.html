Introduction to Termux: Turn Your Android into a Linux Terminal

Termux is a powerful app that turns your Android phone into a Linux terminal. You can run commands, install tools, and even code on your phone. This guide will help you get started with Termux in simple steps.

Step 1: Install Termux

  • Go to the Google Play Store or F-Droid.
  • Search for Termux and install it.
  • Open Termux and grant storage permissions when asked.

Step 2: Update Termux

  • In Termux, type pkg update && pkg upgrade and press Enter.
  • Wait for the update to finish. This keeps Termux up-to-date.
  • If asked to confirm, type y and press Enter.

Step 3: Learn Basic Commands

  • ls: Lists files and folders in the current directory.
  • cd directory_name: Changes to a different folder.
  • mkdir folder_name: Creates a new folder.
  • clear: Clears the screen.

Step 4: Install a Tool (Example: Python)

  • Type pkg install python and press Enter.
  • Wait for Python to install.
  • Test it by typing python --version.

Step 5: Explore More

  • Try commands like whoami or pwd.
  • Install other tools like git, nano, or wget.
  • Use Termux to code or run scripts on your phone.

Tip: Join Termux communities on Telegram or Reddit to learn more tricks!
